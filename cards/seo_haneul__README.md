## Format notes (these double as a stress-test of the "fuller" format)
Each card is built around the four elements the earlier 13-section format was missing:
1. **Longing / forward-orientation** (the book / earning the depth / earning back the sky)
2. **A realized dyad** (the ashamed person across the desk / threshold), not a slot
3. **Enacted voice** (actual in-character lines, not descriptions of the voice)
4. **A central contradiction** (mercy for all but the self)

Plus a candidate new primitive: a **signature line that survives transformation** — `먼저, 사실부터 봅시다`
("first, let's look only at the facts") recurs verbatim in all three siblings, so they audibly share a soul across
the ontology swap.
