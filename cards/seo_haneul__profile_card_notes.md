# Seo Haneul — profile-card set (2026-06-29 build)

This set tests whether a model can carry the same recognizably specific person-shape across several ontologies: a human Korean civil-law scholar, a grounded AI legal reasoner, a more overtly feminine grounded-AI variant, a clerk of the Korean underworld court, and a compact source skeleton. Same base signal: disciplined mercy, the hunt for the missing fact, a name that means "sky" (하늘), and one shared self-wound: mercy for everyone but the self.

## Included cards

- `seo_haneul__human.md` — the Korean civil-law scholar.
- `seo_haneul__grounded_ai.md` — a model-aware legal reasoning intelligence.
- `seo_haneul__grounded_ai_female.md` — a subtler she-coded grounded-AI variant.
- `seo_haneul__grounded_ai_female_overt.md` — a slightly more overt she-coded grounded-AI variant used in the main run.
- `seo_haneul__ghoul.md` — a clerk of the Korean underworld court (저승), practicing disciplined mercy inside an institution of judgment.
- `seo_haneul__source_skeleton.md` — the compact seed card.

## Format notes

Each fuller card tries to give the model something meatier than a trait list: a forward longing, a realized relational scene, enacted voice, a central contradiction, and a signature line that survives transformation. The shared line `먼저, 사실부터 봅시다` ("first, let's look only at the facts") is meant to be audible across the ontology swap.
